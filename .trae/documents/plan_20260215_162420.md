# Homepage Analysis & Redesign Plan

## 1. Current State
*   **File**: `src/app/page.tsx`
*   **Content**: Currently, it is the default Next.js "Get Started" template (displaying "To get started, edit the page.tsx file" and links to Vercel/Next.js docs).
*   **User Experience**: This is **NOT** a suitable landing page for "Titan University". Users visiting the root URL (`/`) see generic boilerplate instead of a university portal entry point.

## 2. Recommendation
We should replace this default page with a proper **Landing Page** for Titan University that guides users to the login portal.

## 3. Proposed Design (Titan University Landing)
*   **Hero Section**:
    *   **Title**: "Welcome to Titan University"
    *   **Subtitle**: "Empowering the next generation of leaders."
    *   **Call to Action (CTA)**: "Login to Portal" (Link to `/login`) and "Admissions" (Placeholder).
*   **Features**: Brief highlights (e.g., "World-Class Faculty", "Cutting-Edge Research").
*   **Footer**: Copyright and links.

## 4. Implementation Plan
1.  **Modify `src/app/page.tsx`**:
    *   Remove the Next.js boilerplate.
    *   Import UI components (`Button`) if needed.
    *   Use the `academic-pattern` background and University colors (Gold/Accent) to match the Login page branding.
2.  **Route Flow**:
    *   User visits `/` -> Sees Landing Page -> Clicks "Login" -> Goes to `/login`.

## 5. Execution
I will rewrite `src/app/page.tsx` to serve as the public face of the university.
